<DESCRIPTION>
The current program achieves macro-F1=0.60 with severe class F1=0.42, indicating poor calibration of probability estimates for the minority severe class. The top-performing programs (score=0.6551) show that wrapping the RandomForest pipeline with CalibratedClassifierCV using isotonic regression is the single most impactful improvement, boosting severe F1 from 0.47 to 0.52.

The key insight is that RandomForest probability estimates are systematically biased for minority classes. Isotonic regression corrects these probabilities non-parametrically, allowing the soft-voting ensemble to properly weight severe class predictions. This must be applied to the RF pipeline before ensemble combination.

We will modify the train_and_predict function to:
1. Import CalibratedClassifierCV
2. Wrap the RF pipeline with calibration using method='isotonic' and cv=3
3. Maintain the existing GB pipeline without calibration (GB probabilities are already well-calibrated)
4. Use soft voting with calibrated RF probabilities

This change directly addresses the severe class under-prediction (pred_dist shows 157 vs actual 164) and aligns with the proven best practices from the top-performing programs.
