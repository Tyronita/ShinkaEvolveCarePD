<DESCRIPTION>
The current architecture relies on handcrafted gait features but misses key dynamics for classifying severe PD (class 2). The original recipe identified this bottleneck and recommended adding Category I (jerk), Category K (trunk stability), and Category L (arm swing) features — which are partially missing here. 

Specifically:
- Jerk (3rd derivative) captures bradykinesia and movement smoothness loss — absent in current feature set.
- Trunk stability (spine joint variability) is a key compensatory control marker in severe PD.
- Arm swing magnitude and asymmetry from shoulder/elbow are hallmarks of PD progression.

These features directly address the 40+ severe→normal and 58+ severe→mild misclassifications by targeting within-walk deterioration and rigidity patterns. Adding them expands the feature vector by 144 (jerk) + 18 (spine velocity) + 36 (arm swing) = 198 dimensions, all derived from the robust 6D rotation representation.

Furthermore, percentile features (10th, 25th, 75th, 90th) on joint velocities improve robustness to outliers and capture gait variability — another severe PD indicator. These were recommended in the original guidelines but missing.

This edit implements all at once to maximize the jump in class 2 F1 — the current bottleneck at 0.47.
