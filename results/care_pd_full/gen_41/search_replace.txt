<DESCRIPTION>
Add temporal segmentation features to capture within-walk gait deterioration and freezing episodes. The current handcrafted features are computed on the entire sequence, which averages out important temporal dynamics like gait freezing or deterioration in moderate-to-severe PD. By splitting each walk into 4 non-overlapping segments and computing per-segment statistics, we can capture the variability of gait patterns within a single walk. This is particularly important for the mild/severe boundary where 58 severe cases are currently misclassified. We compute the standard deviation of per-segment means across all feature categories, adding 144 dimensions of temporal variability that directly proxy the within-walk fluctuations characteristic of moderate-to-severe PD. This change targets the primary bottleneck of severe class F1 by adding discriminative features for gait instability and freezing episodes that are invisible to whole-sequence statistics.

def extract_gait_features(pose: np.ndarray) -> np.ndarray:
    """
    Extract comprehensive handcrafted gait features from SMPL pose sequence.
    Returns a 1D feature vector capturing PD-relevant gait characteristics.
    """
    T = len(pose)
    rot6d = smpl_to_6d(pose)  # (T, 25, 6)
    flat = rot6d.reshape(T, -1)  # (T, 150)

    features = []

    # 1. Global rotation statistics per joint (mean, std, percentiles)
    features.append(flat.mean(0))       # (150,)
    features.append(flat.std(0))        # (150,)
    features.append(np.percentile(flat, 25, axis=0))  # (150,)
    features.append(np.percentile(flat, 75, axis=0))  # (150,)

    # 2. Temporal segmentation: split into 4 segments and compute variability
    # This captures gait deterioration and freezing episodes
    n_segments = 4
    seg_size = T // n_segments
    if seg_size > 0:
        seg_means = []
        for i in range(n_segments):
            start_idx = i * seg_size
            end_idx = start_idx + seg_size if i < n_segments - 1 else T
            seg_means.append(flat[start_idx:end_idx].mean(0))
        # Add std of segment means as temporal variability feature
        features.append(np.std(seg_means, axis=0))  # (150,) temporal variability
